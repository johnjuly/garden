
- 罗马数字10用x来表示，v表示5.张开的手，两个手就是10。象形。vv.两手相对叠合。
-

| 数字  | 符号来源        | 解释   |
| --- | ----------- | ---- |
| I   | 单竖线         | 1    |
| V   | 张开手形        | 5    |
| X   | 两个 V 相对     | 10   |
| L   | 半个 C        | 50   |
| C   | centum（100） | 100  |
| D   | 半个 M（CↃ）    | 500  |
| M   | CↃ 或 ⊃⊂ 形   | 1000 |
